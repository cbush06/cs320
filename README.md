# CS-320 Tutorial

This repository accompanies a 3 part video series I created for my SNHU CS-320 class. The videos walk students through creating a Gradle project in Eclipse, adding necessary dependencies, and coding a simple project similar to the one they're required to build for their Project 1.

The students projects must create simple entities, service classes using an in-memory database (a Java HashMap), and corresponding unit tests. Unlike the PDF tutorial provided by the CS-320 course template, this tutorial uses the Java Validation Annotations (JSR-303) API and the Hibernate implementation.
